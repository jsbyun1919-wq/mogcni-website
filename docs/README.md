# MOG Consulting

MOG Consulting의 콘텐츠 허브 사이트입니다.

## 파일 구조
```
├── index.html          # 메인 허브 (서비스, 진단, 교육, 인사이트, 블로그 미리보기, 문의)
├── about.html          # 회사소개 (등대 메타포, Mission/Vision, 운영원칙, 핵심가치, 팀)
├── diagnostic.html     # 진단도구 안내 페이지
├── style.css           # 공유 스타일시트
├── blog/
│   ├── index.html      # 블로그 목록
│   ├── sales-dependency-structure.html
│   ├── amoeba-prerequisites.html
│   ├── kpi-not-working.html
│   ├── consulting-in-ai-era.html
│   ├── japanese-subsidiary-challenges.html
│   ├── startup-late-system.html
│   ├── sales-productivity-plateau.html
│   ├── strategy-execution-gap.html
│   ├── amoeba-philosophy-not-accounting.html
│   └── consulting-report-in-drawer.html
├── sitemap.xml
├── robots.txt
├── netlify.toml        # Netlify 설정 (한글→영문 리다이렉트 포함)
└── README.md
```

## 배포
Netlify에 이 폴더를 드래그앤드롭하면 바로 배포됩니다.

## 기술 스택
- 순수 HTML/CSS/JS (프레임워크 없음)
- Pretendard 폰트 (CDN)
- Google Apps Script 연동 (문의 폼)
- Google Search Console 인증 메타 태그 포함
